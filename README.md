# random-function

A really awesome random function

## Requirements

You need to have the following services installed in your computer:

* node > 7.6

### Mac OS

```
$ brew install node
```


### Clone the repo

```
$ git clone git@github.com:alexisllamas/random-function.git
```

### Run the app

```
$ yarn start
```

or you can use

```
$ npm start
```

### Run test!

```
$ yarn test
```

or you can use

```
$ npm run test
```

## Tech Stack

* Node - run the JavaScript
* jest - tests
